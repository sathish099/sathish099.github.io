# Zomoto — Food Delivery Landing Page Clone

A responsive food delivery platform landing page inspired by Zomato.

Built by: Sathish (@sathish099)

## Features

✅ Restaurant search & filtering
✅ Category browsing
✅ Responsive mobile design
✅ Modern UI with smooth interactions
✅ Dynamic card rendering

## How to View

1. Open `index.html` in your browser
2. Or run a local server: `python -m http.server 8000`
3. Navigate to `projects/zomoto/`

## Technologies

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

## Project Learnings

- Responsive design patterns
- DOM manipulation
- Search/filter functionality
- CSS Grid layouts
- Mobile-first approach
