# YTube — YouTube-style Landing Page Clone

A lightweight YouTube-inspired video streaming platform clone.

Built by: Sathish (@sathish099)

## Features

✅ Video card grid with thumbnails
✅ Search & category filtering
✅ Modal video playback (YouTube embeds)
✅ Responsive mobile design
✅ Clean, modern UI

## How to View

1. Open `index.html` in your browser
2. Or run a local server: `python -m http.server 8000`
3. Navigate to `projects/youtube/`

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Modal)
- Vanilla JavaScript

## Project Learnings

- Modal implementation
- Video embed integration
- Responsive grid layouts
- Filter & search functionality
- Event handling
